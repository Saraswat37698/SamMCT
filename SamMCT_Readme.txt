# Samsung Promotional React App

This is a promotional app built using React.js that showcases various Samsung products and features. The app includes different sections such as product highlights, TV/AV details, recommended products, and more. The user can toggle between Light and Dark modes and navigate through different sections with smooth animations.

## Features

- **Light/Dark Mode Toggle**: Switch between light and dark modes for a better user experience.
- **Product Carousel**: A dynamic image carousel to display promotional product images.
- **Product Highlights**: Features Samsung's flagship products with details like pricing and offers.
- **Mobile Posters**: Displays images of the latest Samsung mobile devices with promotional text.
- **TV & AV Section**: Showcase different Samsung TV models and AV products.
- **Recommended Products**: A sliding section showcasing recommended Samsung products that can be bought.
- **Explore Section**: Discover new Samsung features and technologies.
- **Search Functionality**: A search bar to look for products across the website.
- **Navigation**: Easy-to-use navigation to browse various product categories and services.
- **Footer**: Contains important links to product categories, support, community, and sustainability.

## Technologies Used

- **React.js**: The core JavaScript library for building the user interface.
- **CSS**: Styling for the layout and components.
- **React Hooks**: For managing state and effects.
- **Smooth Animations**: Implemented using CSS transitions for smooth user interactions.
- **Responsive Design**: Ensures the app looks great on all screen sizes.
